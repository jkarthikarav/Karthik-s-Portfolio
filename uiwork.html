Onboarding Flow – BNPL App Concept

PayDaze is a conceptual Buy Now, Pay Later (BNPL) app that allows users to split payments into installments.
The UI design images below depict the onboarding process.

Onboarding Flow
1. Welcome – 3 intro screens orient users to PayDaze.
2. Profile Setup – Enter personal details + create account.
3. Identity Verification – Upload ID for KYC validation.
4. Bank Linking – Connect bank account/credit/debit card.
5. Terms – User reviews + accepts policies
On Completion, they will access the Dashboard
