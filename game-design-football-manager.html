Project Goals

Football Manager has a lot of depth, but the interface can feel messy and hard to use. I looked at three key screens on the club page: Club Overview, Registration, and Transfer History.

These are all part of the tabs and dropdowns you see when viewing any club. The Club Overview is what you see first.

I asked one main question: How can I show key club info without crowding the screen? 

My redesign focuses on grouping related info together which helps users scan faster, make decisions quicker, and feel more in control when they view a club. My redesign focuses on how other clubs are viewed, which is why the user playing as Liverpool FC is looking at a Chelsea FC Club Overview. While the game does generate an in-game next match report for the club, the user should still be able to view the club overview screen and make decisions for themselves, i.e., to not have to rely on the package that the user's in-game assistant manager generates, which might contain errors and the personal opinions of the assistant.

The Redesigned screens also use latest player and club data, but was originally designed with the 2021 version of the game in mind. This is because redesigns are usually based off of the fandom concept of Mods, a process wherein fans create customised skins, icon packs, game data (such as using 2023 data in a 2020 version, 1999 data in a 2015 version, etc) and other content for the game on their localised PC using code and the in-built FM editor. This adds a level of customisation and close-ness that the released version can't provide.

Sports Interactive, the developer of FM, does not officially sell in-game skins. The game provides a default skin, and players are encouraged to customize their experience through community-created mods. FM's modding community is vibrant and creative, producing a wide array of custom skins, face packs, logos, and other graphical enhancements.

The community's active participation shows a strong demand for aesthetically and game-value enhancing add-ons.

Conclusion: Thoughts and Takeaways

Redesigning the Football Manager interface was all about striking the right balance between presenting detailed data and ensuring the layout was visually accessible and easy to digest.

My design focuses on optimizing the visual flow, organizing information in a way that makes it easy for users to scan and understand at a glance. I placed a strong emphasis on visual hierarchy and grouping related data to make it intuitive and less overwhelming.

Also, I envisioned it as an AI-powered interface: its recommendations and event highlights would adapt based on user preferences and behaviour data, improving relevance over time.

Key Takeaway: Designing and Presenting Data in Games with Purpose.

Every piece of information on screen should either enhance realism or serve the decision-making process. In complex systems like Football Manager, the interface must focus on what’s truly needed, ensuring that every element contributes meaningfully to the player's experience.
