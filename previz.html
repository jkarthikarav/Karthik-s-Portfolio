Project Context

This was a hybrid project at Fix It In Post Studios: a deliverable for a Client and a test for our internal PreViz system.

Help a director showcase their story to producers, while designing an AI-assisted pipeline we could reuse across productions.

Design Touchpoints & Decisions

Designed a layered pass system (BG, FG, Characters)

The system was structured with downstream editing in mind, allowing animators to composite each element independently as required.

Refined asset quality through manual Photoshop cleanup

This included refining edges, adjusting color consistency, and ensuring each asset was ready for animation.

Built shot assignment workflows and templates

Our workflow included layered file templates, helping each designer know exactly how to structure and export assets.

Set naming conventions, export structures

Consistent folder structures and clear file labeling made it easy for the editor to animate quickly without back-and-forth clarification or confusion.

The Process in a Nutshell

We delivered a working AI-assisted PreViz system, designed to visualize film scenes quickly using modular shot breakdowns, hand-drawn characters, and layered passes.

Client-Editor Reference Session

The director and editor selected key visual references together, establishing a clear tone, pacing, and visual direction for the story pitch.

Scene Breakdown & Asset Assignment

The visual references were split into scenes and shots. Each was structured for layering (background, foreground, characters) and assigned to our team accordingly.

Green was to be sketched, Red was to be removed, the rest of the image was to be generated in the style the Director preferred.

Character Sketching

Our sketch artist hand-drew main characters (hero, heroine, and other key figures) based on the director’s vision.

Scene Generation

The design team and I generated background and foreground layers for each shot using the references. I cleaned and refined all assets to ensure consistency.

Layered Pass Compilation & Export

For every shot, I compiled and exported background, foreground, and character passes as separate files.

Final Animation by Editor

The editor assembled and animated the passes, bringing the scenes to life.

What I Learned

This project taught me how real-world creative constraints like tight budgets and fast timelines are the best proving ground for building workflows and information systems.

Unlike most AI/Tech-guru sponsored workflows, by settling upon the right mix of control we provide AI tools with over our professional lives, we delivered faster without sacrificing human creative control.

Key Takeaway: Prototyping inside real projects accelerates adoption

Testing the AI-assisted PreViz workflow during a real client project made our pipeline faster to validate and immediately useful.
